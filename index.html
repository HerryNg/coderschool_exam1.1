CLI

What is CLI?

CLI is command line interface. It can allow the programmers to text commands instructing the computer to specific tasks.

What is the advantage/disadvantage of CLI?

Advantage: -Fast performance: programmers can perform many tasks by entering the commands in the CLI. -Less memory: CLI use less memory when we compare to GUI. -Do not required window: CLI can perform not only in Window, but it also can perform all operating systems such as Mac, and Linux. -Autocomplete: When we type a command, most of the command will auto complete bu simply clicking TAB button.

Disadvantage Difficult to remember: CLI so many command lines that causes difficulties for users. Redo command: users cannot redo command line interface. Users have to carefully use command line.

Which do you prefer, CLI or GUI (Graphical User Interface)? Why?

I prefer CLI to GUI. The reasons are CLI is faster and consume less memory.
